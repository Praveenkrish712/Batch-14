HTML - Hyper Text markup Language

* To create web pages
* Not a Programming Language
* its a markup Language
* Every Website starts with HTML

1) Heading - h1 to h6

2) Paragraph - p

3) hr 

4) br

5) Text formatting - i, b or strong, mark, sub, u and sup

6) Anchor tag (Linking Purpose) - a

7) image tag - img

8) list - Order list and unorder list

9) tables - tr, th and td

10) Forms - input, label, placeholder, required

11) Media Elements - Audio and Video

12) Sematic Tags - header, footer, nav and section
